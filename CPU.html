Central Processing Unit


The Centeral Processing Unit or CPU is the brain of the computer most of the computers functions are carried out here. The CPU is responsible for executing a sequence of stored instructions,
This can include stored numbers and command lines as well as the computer applications. The CPU is also reponsible for taking inputs form an input device and creating a output for it.
All CPU's have three main features, which are: Cores, Clock Speed and Cache
A CPU can contain one or more processing units. Each unit is called a core. A core contains an ALU or an Arithmatic Logic Unit. Most modern day CPU's have Dual or Quad Cores, High end CPU's can evn hold anywhere from six to eight.
This allows the user to multitask programs.
The clock speed indicates how fast the CPU can run, this is measured in megahertz (MHz) or gigahertz (gHz) and means how many instruction cycles the CPU can complets with in a second.
Finally Cache is a small amount of memory which is a part of the CPU. It is used to temporarily hold instructions and data that the CPU is likely to reuse.
